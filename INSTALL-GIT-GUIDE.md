# 📥 INSTALL GIT - STEP-BY-STEP GUIDE

## 🔧 YOUR SYSTEM NEEDS GIT

Git is not installed on your computer. You need it to push code to GitHub.

---

## ✅ INSTALL GIT (EASY - 2 MINUTES)

### Step 1: Download Git
1. Go to https://git-scm.com/download/win
2. Click the download link for Windows (64-bit recommended)
3. Wait for download to complete

### Step 2: Run Installer
1. Find the downloaded file (usually `Git-X.X.X-64-bit.exe`)
2. Double-click to run it
3. Click **Next** on each screen (use defaults)
4. Near the end, make sure **"Git Bash Here"** is checked
5. Click **Install**
6. Click **Finish**

### Step 3: Restart PowerShell
1. Close PowerShell completely
2. Open PowerShell again
3. Run: `git --version`

**If you see a version number, Git is installed! ✅**

---

## 🚀 QUICK START AFTER GIT INSTALL

Once Git is installed, run these commands:

```bash
cd d:\face-attendance
git add .
git commit -m "Fix: Optimize dependencies for Railway deployment"
git push origin main
```

Done! Railway auto-deploys 🎉

---

## 🆘 IF INSTALLER DOESN'T WORK

### Alternative: Use Windows Store
1. Open Windows Store (or Microsoft Store app)
2. Search for "Git"
3. Click on official Git
4. Click **Get** or **Install**
5. Wait for installation
6. Restart PowerShell

---

## ✅ VERIFY GIT IS INSTALLED

After installation, close PowerShell and open a new one. Run:

```bash
git --version
```

You should see something like:
```
git version 2.44.0.windows.1
```

If you see this, **Git is installed!** ✅
