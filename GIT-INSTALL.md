## ✅ QUICK FIX (3 STEPS)

### Step 1️⃣: Install Git
- Go to: **https://git-scm.com/download/win**
- Click download (64-bit for Windows)
- Run the installer
- Click **Next** on every screen
- Click **Finish**

**Time: 2-3 minutes** ⏱️

- Close PowerShell completely
